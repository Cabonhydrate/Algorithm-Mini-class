## 大作业：CIFAR-10图像分类

### 项目概述
使用ResNet18模型对CIFAR-10数据集进行图像分类，实现了完整的训练、评估和可视化流程。
详细介绍请看`Week2/大作业/week2_miniproject.md`

### 主要功能
1. **模块化设计**：将代码拆分为模型、数据、工具、配置等多个模块
2. **数据处理**：自动下载并预处理CIFAR-10数据集，支持数据增强
3. **模型训练**：使用ResNet18模型从头训练，支持学习率调度
4. **评估指标**：计算详细的评估指标，包括TP、FN、FP、TN、准确率、精确率、召回率、F1分数和混淆矩阵
5. **日志记录**：将训练过程和评估结果保存到日志文件
6. **模型保存**：自动保存最佳模型权重
7. **结果可视化**：生成混淆矩阵等可视化结果

### 训练结果
- **准确率**：0.8737
- **精确率**：0.8736
- **召回率**：0.8737
- **F1分数**：0.8735

### 运行说明
1. **训练模型**：`python 大作业/train.py`
2. **评估模型**：`python 大作业/evaluate.py --model_path 大作业/checkpoints/best_train.pth`
3. **可视化结果**：`python 大作业/visualization/plot_eval.py --metrics_json 大作业/logs/metrics_eval_20260206_132455.json`
